# 브랜드 논란 데이터 분석 프로젝트
SNS 논란이 브랜드 신뢰도에 미치는 영향을 분석하는 프로젝트입니다.

# 프로젝트 목적
- 질문: 논란 시 소비자 반응의 ‘순서’와 ‘패턴’은 무엇인가?
- 지표: [감정(부정비율) → 검색량 → 리뷰수 → 팔로워]
- 목표: 전파 경로(감정→검색→행동)와 시차(lag) 계량화

## 폴더구조
- data/raw : 원본 데이터
- data/processed : 전처리 후 데이터
- notebook : 분석용 주피터 노트북
- scripts : 수집/전처리 코드
- reports : 결과 요약 및 인사이트
- figures : 시각화 이미지
